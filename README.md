# 🏟️ Smart Stadium Assistant

> An AI-driven crowd management and real-time coordination dashboard for large-scale venues.

![Smart Stadium Assistant Preview](https://img.shields.io/badge/Status-Hackathon_Ready-success) ![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react&logoColor=61DAFB) ![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat&logo=vite&logoColor=white) ![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=flat&logo=tailwind-css&logoColor=white)

The **Smart Stadium Assistant** enhances the stadium experience by optimizing crowd movement, reducing wait times, and enabling real-time operational coordination. Instead of manually navigating congested areas, attendees and staff are provided with proactive, data-driven routing recommendations based on simulated live telemetry.

## ✨ Core Features

*   **🧠 Intelligent Routing Engine:** Calculates the most efficient path to gates, food stalls, and washrooms by heavily weighing real-time crowd density against estimated queue times.
*   **⚡ Proactive Smart Alerts:** Monitors the network for sudden crowd surges and automatically issues redirection alerts to less congested alternatives.
*   **📊 Live Wait Time Telemetry:** Dynamic progress trackers that isolate and highlight the absolute fastest options for amenities in real time.
*   **🗺️ Interactive Density Mapping:** A visual heat-map representation of stadium zones (North, South, East, West, and Pitch) allowing quick bottleneck identification.

## 🛠️ Technology Stack

*   **Frontend Framework:** React 18
*   **Build Tool:** Vite (for lightning-fast HMR and optimized production bundles)
*   **Styling:** Tailwind CSS (Dark mode optimized, glassmorphism UI)
*   **Architecture:** Component-based modular ES6 structure

## 🚀 Running Locally

To run this project on your local machine, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/Smart-Stadium-Assistant.git
   cd Smart-Stadium-Assistant
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the App:**
   Open your browser and navigate to the localhost URL provided in your terminal (usually `http://localhost:5173`).

## 🚢 Deployment

This project is fully optimized for single-click deployment on **Vercel**. 
Simply import the GitHub repository into your Vercel dashboard, and it will automatically detect the Vite build configuration. No manual setup required!
